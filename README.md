# 📸 FotoApp para Ana ✨

¡Bienvenida a tu propia aplicación personalizada, Ana! Esta es una **Demo App** creada con **Streamlit** para capturar momentos especiales, aplicar filtros mágicos y guardar recuerdos inolvidables directamente desde el móvil o el ordenador.

---

## 🌟 Características principales

- 📷 **Captura en tiempo real**: Usa la cámara de tu dispositivo para hacerte fotos al instante.
- 🎨 **Filtros Mágicos**: Elige entre varios estilos para tus fotos:
  - **Original**: Tal cual eres.
  - **Blanco y Negro**: Un toque clásico y elegante.
  - **Sepia**: Estilo vintage de película antigua.
  - **Invertir**: ¡Colores locos y divertidos!
- 💾 **Descarga Directa**: Guarda tus creaciones favoritas con un solo clic.
- 📱 **Diseño Responsive**: Funciona perfectamente en iPhone, Android y PC.

---

## 🛠️ Tecnologías utilizadas

Esta app utiliza el poder de la Inteligencia Artificial y el desarrollo rápido de Python:

*   [Streamlit](https://streamlit.io/) - El framework más rápido para crear apps de datos.
*   [Pillow (PIL)](https://python-pillow.org/) - Para el procesamiento y edición de imágenes.
   [NumPy](https://numpy.org/) - Para los cálculos matemáticos de los filtros.
